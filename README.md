## Project information
ChatApp1 is a Java application that shows how basic user login systems work. It lets users register and log in by checking if their username, password, and phone number follow certain rules.
The system allows users to register and log in by validating their username, password, and phone number according to predefined rules.
It ensures that user input follows correct formatting standards, improving data reliability and application security.

---
## Features
### USERNAME VALIDATION
It checks if the username meets the requirements(Must contain an underscore(_) and must not exceed 5 characters)
### Password Validation
it checks if the password meets the requirements(At least one capital letter,one number,one special character)
### Cellphone Number Validation
check if cellphone Number meets requirements(Must include country code (e.g.+27) and Must be valid numeric format)
